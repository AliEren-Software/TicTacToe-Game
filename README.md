# TicTacToe-Game
2D 4x4x4 TicTacToe Game 

For play the game you can run TicTacToe/src/Main.java


How to play:
------------

In one plane, classic rule : If 'X' or 'O' orders same horizontally or vertically by 4 which do that will win.
